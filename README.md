# 🐭 Mouse on the Train

**Mouse on the Train** — это 2D-платформер с элементами RPG, разработанный на языке C++ с использованием графической библиотеки SFML. Игрок управляет мышью, которая путешествует на поезде, сражается с монстрами, лечится мармеладом и добирается до конечной станции.

Проект выполнен в рамках курсовой работы по дисциплине «Программирование на языках высокого уровня»

## 🎮 О проекте

**Mouse on the Train** — это игра, в которой вы управляете мышью, путешествующей на поезде. Ваша цель — пройти через все вагоны, победить монстров и добраться до конечной станции.

**Ключевые особенности:**
- 🐭 Управление мышью: движение, прыжки, атака
- 🩸 Система здоровья и лечения мармеладом
- 👾 4 уникальных монстра с разными характеристиками
- 🚪 Переход между локациями через двери
- 🏆 Таблица рекордов (топ-4 лучших времени)
- 🎨 Анимации персонажей (покой, шаг, атака, прыжок)
- ⌛ Таймер прохождения

---

## 🕹️ Игровой процесс

Игра состоит из **7 локаций**:

| Локация | Описание |
|---------|----------|
| Станция отправления | Обучение, подсказки по управлению, вход в поезд |
| Вагон 0 | Проходной вагон без монстров |
| Вагон 1 | Кондуктор (45 HP, 12 урона) |
| Вагон 2 | Фокусник (60 HP, 15 урона) |
| Вагон 3 | Спекулянт (50 HP, 10 урона) |
| Вагон 4 | Финальный босс (100 HP, 20 урона) |
| Станция прибытия | Финал, запись рекорда |

**Механики:**
- После победы над монстром вы получаете **+1 мармелад** и открывается дверь
- Мармелад восстанавливает **10 HP** (клавиша R)
- Атака наносит **15 урона** (клавиша Q)
- При поражении игра заканчивается

---

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `A` / `D` | Движение влево / вправо |
| `Space` | Прыжок |
| `Q` | Атака |
| `R` | Лечение мармеладом |
| `F` | Взаимодействие с дверью |
| `F11` | Полноэкранный режим |
| `Esc` | Выход из игры |

## 🛠️ Технологии

- **Язык:** C++17
- **Среда разработки:** Microsoft Visual Studio 2026
- **Графическая библиотека:** SFML 2.6.2 (Simple and Fast Multimedia Library)
- **Стиль кода:** Объектно-ориентированный, модульный

---

## Системные требования

- ОС: Windows 10 / 11 (64-bit)
- Процессор: 1.5 GHz
- ОЗУ: 2 GB
- Видеокарта: с поддержкой OpenGL 2.0
- Разрешение экрана: 1920x1080 (рекомендуется)
